Información general del proyecto

Este documento presenta un resumen de los diferentes documentos y enlaces que son parte de la información general del proyecto. Todo este contenido es generado automáticamente por Maven en nombre de este proyecto.

Resumen

Documento Descripción
Gestión de Plugins Este documento presenta los plugins que se han definido mediante la sección pluginManagement.
Gestión de Distribución Este documento proporciona información sobre la gestión de distribución de este proyecto.
Información de Dependencias Este documento describe cómo incluir este proyecto como una dependencia utilizando distintas herramientas de gestión de dependencias.
Repositorio de Código Este es un enlace al repositorio de código que puede ser consultado mediante un navegador web.
Listas de Correo Este documento proporciona información para suscribirse y acceder a los archivos de la lista de correo del proyecto.
Gestión de Incidencias Enlace al sistema de gestión de incidencias de este proyecto. Incidencias (errores, características, cambios) se pueden crear y consultar usando este enlace.
Integración Continua Enlace a las definiciones de los procesos de integración continua que construyen y prueban el código de forma frecuente y regular.
Plugins del Proyecto Este documento presenta los plugins para la construcción y generación de informaes usadas por el proyecto.
Licencia del Proyecto Enlace a la definición de las licencias del proyecto.
Equipo del Proyecto Este documento proporciona información sobre los miembros del proyecto. Estas son las personas que han contribuido al proyecto de una forma u otra.
Resumen del Proyecto Este documento presenta la información relacionada con el proyecto
Dependencias Este documento presenta las dependencias del proyecto y proporciona información sobre cada dependencia.
